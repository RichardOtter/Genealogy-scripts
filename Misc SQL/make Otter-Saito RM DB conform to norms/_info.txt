Steps to change the Otter-Saito RM database so that it conforms with
GEDCOM and standard genealogical data norms



File name= Convert Confidence from S to E.sql
Script changes all RM dates that use "say" confidence level to "est"
confidence level to better conform to GEDCOM v5.5


File name= Delete all places with uncertainty markers.sql
Deletes all places that start with ~ or [~] or [~|]
and deletes all uses of those placeTable entries.

File name= Remove bracketed text in NameTable.sql
Remove text within [] in the names table All fields.

File name= Remove bracketed text in PlaceTable.sql
removes bracketed text.
Probably does not cause genealogical data loss.

File name= Drop non-RM-standard tables.sql
Drop Aux tables.
Drop Look-Up tables and views
